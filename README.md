<h1>自學筆記</h1>
這是我在自學的過程中所整理下來的筆記，以便往後可以快速參考與複習。<br>
主要是以 Wes McKinney 與 Michael Heydt 的書為主，遇到比較難理解的地方除了 Google 之外，也會找尋影片幫助理解。<br>
Michaek Heydt的書比較偏基本功，Wes McKinney的書比較偏實際應用面。搭配之下學起來相當扎實。<br>
兩本書都相當推薦!
